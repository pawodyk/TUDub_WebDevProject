Welcome to my website about Web Technologies

For a last decade the web development has grown from very simplistic websites that used tables to position the element, to modern, responsive pages using flex positioning and embedded multimedia element. The new design is clearer and easier to navigate, and made the web pages become something more than just hyper text documents. HTML5 also improved on semantics making sites more approachable for text readers and visually impaired.
