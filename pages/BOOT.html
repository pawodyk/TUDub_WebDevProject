About Bootstrap

Bootstrap is a open-source frontend framework, developed originally by Twitter employees Mark Otto and Jacob Thornton. It was developed to encourage consistency between sites. It helps to prototype and build new sites with ease and consistent looks. And you use it by adding predefined bootstrap classes to your tags to apply the styling.

To start using a Bootstrap you need to link it to your html page using link tag, just like any other stylesheet: <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

You can also use their predefined JavaScript snippets. For more info visit Getting Started section of the documentation.

Bootstrap Documentation

The current Bootstrap version is 4.3 and a full documentation is available on their official website.

The Bootstrap is free and open-source and fully available on GitHub.
