Basics of HTML

HTML stands for Hypertext Markup Language. It is the most widely used markup language to write Web Pages. It is responsible for controlling the structure and semantics of web page.

Markup languages use tags to "mark-up" a content of the document inside them, using tags. A paragraph, for example, is marked with <p> and closed with </p>, this indicates to the web browser that this section is a paragraph. <p>This is a basic paragraph of text in html</p>

While Hypertext refers to the way in which HTML documents are linked internally and externally, using the anchor tag for example: <a href="http://www.google.com">Link to Google</a> The above example links to the Google search page and shows how we can link one web page to another using hypertext. In the above example we can also observe use of attribute href="".

Short origin story of HTML

Initially HTML was developed to allow researchers and academicians to share a scientific research and documentation, focusing mostly on standard document elements like headers, paragraphs and floated images. With the advent of World Wide Web, HTML evolved and become widely used to format web pages with the help of different tags available in HTML language, allowing for Embedding the Multimedia.

HTML Syntax

The example of very basic HTML page:

<!DOCTYPE html>
<html>
        
    <head>
        <title>Here goes your page title</title>
    </head>

    <body>
        <!-- Comment Example -->
        <h1>Here goes your heading</h1>
        <p>Here is a paragraph</p>
    </body>

</html>

In the above example we can see the basic structure of the HTML document with title, heading and paragraph. In Practice the HTML documents consist of many more HTML tags, however using the very basic code we can make a website.

At the top of the page we can see <!DOCTYPE html> this code indicate that this is a HTML5 document and is called Document Type Declaration

Two main sections of the web page are <head> and <body>, this element distinguish to the browser which section should be displayed to the user and which should be read by computer eg. web browser or search engine

The <head> element is holding elements such as:

  • <meta> - which hold metadata (or information) about the HTML document.
  • <title> - shows page name displayed in the web browser tab as well as the name of the bookmark
  • <link> - likes the external stylesheet to HTML document
  • <style> - implements the internel stylesheet to the HTML document
  • <script> - implements the internel or links the external script to the HTML document

The <body> element contains all the information that is displayed to the user. Most of the tags represent the elements of the document like headers, paragraph, images or links. eg.:

<h1>This is a main header<h1>
<h2>This is a sub header<h2>
 ⋮
<h6>This is a least important header<h6>

<p>This is a paragraph<>

<img src="myPhoto.jpg" alt="my Photo">

<a href="https://google.com">This is a link to google website<body>
<a href="#footer">This is an internal link, it will take you to the element with id footer<body>

The html headings are indicated with h1, h2, h3, h4, h5, h6 where the number beside the letter h indicated the importance of the heading, ie. main heading on the page will always be marked by <h1> while sub-heading will be <h2> and so on... This is just like in any document or book.

The paragraph is a most basic element of the page and is marked by <p> tag.

Images are marked with a tag <img> and composuary attribute src="" which points to the image. It is also important to add alt="" attribute that provides the semantic meaning to screen readers and also will display the text when photo could not be displayed.

There are many more tags and HTML5 introduced new ones, to see them all use Mozilla Developer Network or Official World Wide Web Consortium HTML5 specifications

HTML5 Semantic usage

Semantic HTML is a way of adding and enforcing a meaning to the web page instead of just a presentation or looks. It is an important principle of the Web Standards movement and is currently supported by all modern web browsers

Some of the first semantic HTML tags include the header tags <h1> to <h6>, <blockquote>, <code> and <em> as well as the semantic elements introduced by HTML5 such as section, article, footer, progress, nav, aside, mark, and time. There are many more semantic tags available, to learn more about the semantic web and accessibility visit w3 Consortium or articles on Semantics in HTML and Accessibility on Mozzila Developer Network.
