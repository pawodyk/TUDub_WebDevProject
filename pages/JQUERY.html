About jQuery

jQuery is a open-source JavaScript Library. It main use is to help traversing the DOM tree and HTML manipulation, As well as handling events, develop Ajax applications and apply CSS animations. It has easy to use API and is compatible with most browsers.

The simple code example looks as follows: $( "h1.welcome" ).html( "Hello, World!" ); This code will change the html content of heading 1 with class welcome to Hello, World!. As we can see we use $ to invoke the jQuery API and we select elements using the syntax very similar to CSS selectors eg. h1.welcome .

We can also dynamically modify the css value of html elements using $( "p" ).html( "color", yellow ); this code will add the inline css color: yellow; to all paragraphs

jQuery has many more functions, to learn more visit official jQuery API
