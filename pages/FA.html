About Font Awesome

Font Awesome is a icon toolkit based on the CSS. It was developed to work with Bootstrap and provide the icons that are generated like a web font. This allows us to style or resize them just like normal font. For Example the GitHub icon looks like this while displayed inline: using following html code: <i class="fab fa-github-square"></i>.

But we can change the color and size of the icon using CSS, in fact we can use any css styling that would apply to any font element, we can even use transform to rotate the icons:

To view all the icons please visit the official Font Awesome website.
