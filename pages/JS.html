Basics of JavaScript

JavaScript is dynamic, weakly typed, high-level programing language. Together with HTML and CSS it is a cornerstone of the web development. Where HTML is responsible for structure of the document and CSS is responsible for looks, the JavaScript allows us to expand the functionality of the website beyond the simple text document. It is widely used for programing web applications and adding functionality to web pages.

JavaScript is a interpreted programing language and it can be typed in directly into console build in most modern web browsers. In Order to access the Console in Mozilla Firefox or Google Chrome press the F12 key or right click the page and select "Inspect". Next click on the console tab and voilà you can start writing your JavaScript code. For Example console.log("Hello, World!");

JavaSctipt Syntax

To declare variable we simply use var myVariable = "Hello, World"

The five primitive data types in JavaScript are:

  1. String - Hold String of characters
  2. Number - Hold numbers, including special not-a-real-number values like:
    • Infinity Infinity
    • -Infinity Minus Infinity
    • NaN Not a Number (for example a result of dividing 0 by 0)
  3. Boolean - Holds True or False value
  4. Undefined - Uninitialized variable will hold a value of undefined
  5. Null - Null indicates the empty object, in JavaScript it differs from undefined by the fact that it is

It is important to note that Strings in JavaScript are immutable, and also that JavaScript do not distinguish between integer values and floating points, instead it use IEEE-754 standard to represent all numbers as a double floating points.

Arithmetic operations:

// Simple arithmetic operations:
9 + 2 // Addition
9 - 2 // Subtraction
9 * 2 // Multiplication
9 / 2 // Division
9 % 2 // Modulo Division (Reminder)

Declaring Variables

Variable declaration in JavaScript: var myVar1; The Scope of variable declared this way is its current execution content, which means that their scope is within current function or program if they were declared globally. If you wish to declare a variable with a block scope you need to use: let myVar2;

Finally if you wish to declare constant you need to use: const myConst = "Hello, World!" As expected the declared constant value cannot be changed once assigned during execution, and cannot be redeclared. Constants in JavaScript have block scope just like let declaration.

Loops

// FOR LOOP:
for (var i = 0; i < 10; i ++) {
    console.log("Number: " + i);
}

// While Loop:
var count = 10;
while(count > 0) {
    console.log(count);
    count--;
}
console.log("... TAKE OFF");

// Do While Loop:
var x = 0;
do {
  x += 2;
  console.log(x);
} while (x > 5);

// Enhanced For Loop
for (var item in myArray) {
    console.log(item + " in my array");
}
