Basics of SASS

Sass stands for "Syntactically awesome style sheets" and it is a style sheet scripting language that is being compiled to CSS. Sass Extends the functionality of CSS and provides the mechanism that are available in programing languages to help with organizing and writing stylesheets eg. Variables , Loops, Mixins and more...

The basic SASS syntax looks like this: 

p 
    margin: 
        bottom: 10px
        top: 5px
    a
        color: @col-red
        text-decoration: none
  
li
    font:
        family: sans-serif
        weight: bold
        size: 2em

Once completed we compile the Sass code into standard CSS using Sass compiler, so that our code could be understood by the browsers.

SASS vs SCSS

SCSS or "Sassy CSS" is a version of Sass with newer syntax. The main difference is that it uses { } instead of indentation and requires the ; at the end of each declaration more like in standard CSS

p { 
    margin: {
        bottom: 10px;
        top: 5px;
    }
    a {
        color: @col-red;
        text-decoration: none;
    }
}

li {
    font: {
        family: sans-serif;
        weight: bold;
        size: 2em;
    }
}
