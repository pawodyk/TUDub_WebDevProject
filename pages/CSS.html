Basics of CSS

CSS is Cascading Style Sheet, provides the styling to markup language like HTML. HTML in itself do not provide any styling to the document and the "default" styling we see when we display the page without any CSS code is a default browser style provided by the browser (note that it may slightly differ between browsers).

/* Paragraph tag - Selector */
p {
    /* Example of Style declaration consisted of property and value */
    font-family: sans-serif;
}

As shown in the example above the CSS consist of the selector, content block and list of attributes. Selector, as the name indicates, selects the appropriate tag, id or class in HTML. After the selector we open curly brackets { } to start the declaration block.

Inside declaration block we can specify the property followed by : and then attribute value. We close the attribute declaration using ; For example:color: red; which changes the color of the text to red.

There are many CSS properties and each of them uses set of corresponding values, for the full list visit Mozilla Developer Network or w3c Documentation

CSS Selectors

CSS Selectors allow us to select specific tags either by tag type, tag id or tag class as per example below:

/* Example of a comment */

/* Basic selector - selecting all header 1 tags */
h1 { }

/* Children selector - selecting anchor tags inside paragraph */
/* NOTE: the anchor tag doesn't need to be direct child of a paragraph */
p a { }

/* Multiple Selectors - selecting header 1, header 2, and header 3 */
h1,
h2,
h3 { }

/* Class selector - selecting all elements with class box */
.box { }

/* ID selector - selecting element with id logo */
#logo { }

The above example shows only the most basic selectors. There are many more selectors allowing us to target, for example, direct children using >, simblings using + or even all elements using universal selector *. The list of all selectors can be found on Mozilla Developer Network or w3c CSS specification

It is important to note that selectors have predefined specificity which may cause one CSS rule to overwrite other rule. To learn more please see this article at CSS-TRICKS or documentation at Mozilla Developer Network

Pseudo Classes

In CSS we can use pseudo classes to style the HTML element when the specific condition is met, for example when we hover over element, using pseudo class :hover.

We use pseudo classes by entering : directly after the selector (No Spaces!), followed by the name of the pseudo class, as in example below: li:last-child { } Above code will select any element li that is a last child of other element. For a full list of CSS pseudo classes please see MDN Documentation or Official CSS specification
